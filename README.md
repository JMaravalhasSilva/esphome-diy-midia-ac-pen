# ESPHome DIY Midia AC Pen
## An easy to make wifi pen to control midia AC units

This repo contains the files to make a wifi pen to control midea AC units using the [ESPHome Midea component](https://esphome.io/components/climate/midea/).

The pen uses only a handful of components which are extremelly easy to obtain, and extremelly easy to solder as well!

Bill of materials: 
 - x1 ESP32 C3 Super Mini board
 - x2 2N3904 Transistors
 - x4 10kOhm Resistors
 - x1 Male USB connector
